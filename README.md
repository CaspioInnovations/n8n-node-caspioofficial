# n8n-nodes-caspioofficial

This is an n8n community node for integrating with [Caspio](https://www.caspio.com/). Caspio is a low-code platform that helps you create custom business applications fast, combining database management, application building, and workflow automation in a single platform.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.

## Installation

### Community Nodes (n8n Cloud)

For verified community nodes on n8n Cloud:

1. Open your n8n workflow editor
2. Select the **+** button or press **Tab** to open the nodes panel
3. Search for "Caspio" and select it
4. Select **Install** to add the node

### Self-Hosted

For self-hosted n8n instances:

1. Go to **Settings > Community Nodes**
2. Select **Install**
3. Enter the npm package name: `n8n-nodes-caspioofficial`
4. Agree to the risks of installing community nodes
5. Select **Install**

Note: Only the instance owner can install community nodes. Admin accounts can uninstall nodes if needed.

## Documentation

This node provides comprehensive integration with Caspio's REST API:

- **[Credentials Setup](https://howto.caspio.com/integrate-your-apps/integration-with-n8n/n8n-integration-credentials)** - Configure OAuth2 authentication with your Caspio account
- **[Action Node Operations](https://howto.caspio.com/integrate-your-apps/integration-with-n8n/n8n-integration-actions)** - Work with tables, views, directories, files, attachments, and tasks
- **[Trigger Node Setup](https://howto.caspio.com/integrate-your-apps/integration-with-n8n/n8n-integration-triggers)** - Set up webhooks for real-time event notifications

## Compatibility

This node uses Caspio REST API v3.
